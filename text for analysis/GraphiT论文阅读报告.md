## 2. 研究背景与动机

### 2.1 图数据的广泛应用

图结构数据在科学研究中无处不在：

```
┌─────────────────────────────────────────────────────────────┐
│                    图数据的应用领域                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  🧬 生物信息学                                               │
│     蛋白质结构 → 氨基酸序列图                                │
│                                                             │
│  💊 化学信息学                                              │
│     分子结构 → 原子连接图                                    │
│                                                             │
│  🖼️ 计算机视觉                                              │
│     3D形状 → 点云/图表示                                     │
│                                                             │
│  🏥 电子健康记录                                            │
│     患者关系 → 异构图                                       │
│                                                             │
│  📱 社交网络                                                │
│     用户互动 → 社交关系图                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 图神经网络的局限性

传统的图神经网络（GNN）采用**消息传递范式**：

```
GNN的消息传递机制：

Layer 1:    ● → 聚合邻域信息 → ●'
Layer 2:    ●' → 聚合更广邻域 → ●''
Layer 3:    ●'' → ...

问题：
• 信息传播缓慢，需要多层堆叠
• 长距离依赖难以捕捉
• 计算复杂度随层数增加
```

### 2.3 Transformer的潜力

Transformer的**自注意力机制**允许所有节点直接交互：

```
Transformer的信息传递：

Layer 1:    ● ←→ ● ←→ ● ←→ ●  (全连接)
            所有节点直接通信

优势：
• 单层即可实现全局信息交互
• 长距离依赖自然捕获
• 并行计算效率高
```

### 2.4 核心问题

> **Transformer的核心组件是置换不变的，不考虑图的拓扑结构**

```
问题：普通Transformer无法区分以下两种图：

图A:    ●──●──●                    图B:    ●──●
        │                              │     │
        ●──●──●                        ●──●──●
        (三角形)                        (路径)
        
普通Transformer认为它们"相同"（都是3个节点）
```

## 3. 理论基础：谱图分析与图核

### 3.1 图拉普拉斯矩阵

**定义**：对于具有n个节点的图
```
L = D - A

• D = 度对角矩阵（对角线是每个节点的度）
• A = 邻接矩阵
```

**关键性质**：

```
┌─────────────────────────────────────────────────────┐
│ 对于任意向量 u ∈ Rⁿ                                 │
│                                                     │
│ u^T · L · u = Σ (u[i] - u[j])²                     │
│              = 图上"总波动"                          │
│                                                     │
│ 解释：                                              │
│ • 值差异大 → 波动大 → 不平滑                         │
│ • 值差异小 → 波动小 → 平滑                           │
└─────────────────────────────────────────────────────┘
```

### 3.2 特征值分解与频率类比

```
L = Σ λᵢ · qᵢ · qᵢ^T

特征值 λᵢ 的物理意义：
• λ小 → 低频 → 变化平缓 → 整体结构
• λ大 → 高频 → 变化剧烈 → 局部细节
```

**与傅里叶变换的对应关系**：

| 连续信号处理 | 谱图理论 |
|-------------|---------|
| 傅里叶基 e^(iωx) | 拉普拉斯特征向量 qᵢ |
| 频率 ω | 特征值 λᵢ |
| 微分算子 d²/dx² | 拉普拉斯矩阵 L |

### 3.3 正定核函数

通过在谱域应用正则化函数 r(λᵢ)，可以构造正定核：

```
K_r = Σ r(λᵢ) · qᵢ · qᵢ^T
```

**论文使用的两种核函数**：

#### (1) 扩散核 (Diffusion Kernel)

```
r(λᵢ) = e^(-βλᵢ)

K_D = e^(-βL)
```

**直观理解**：
```
扩散过程：
t=0:    在节点A注入物质
t=β:    物质扩散到相邻节点
t=2β:   继续扩散，趋于均匀

K_D(A, B) = 从A扩散到B的物质量
```

#### (2) p步随机游走核 (Random Walk Kernel)

```
r(λᵢ) = (1 - γλᵢ)^p

K_pRW = (I - γL)^p
```

**特点**：
- 稀疏矩阵（只有p步可达节点有非零值）
- 适合编码局部结构

## 4. GraphiT方法

### 4.1 核心思想

```
GraphiT = Transformer + 图结构编码

两个关键组件：
┌─────────────────────────────────────────────────────────┐
│ 1. 基于核函数的相对位置编码                              │
│    → 编码节点间的结构相似性                              │
│                                                         │
│ 2. 子结构特征编码                                       │
│    → 编码局部拓扑信息                                   │
└─────────────────────────────────────────────────────────┘
```

### 4.2 位置编码策略

#### 4.2.1 绝对位置编码 (LapPE)

来自Dwivedi & Bresson (2021)：

```
每个节点分配向量：
[node_position] = [q₂(v), q₃(v), ..., q_{k+1}(v)]

其中 qᵢ 是拉普拉斯特征向量
```

**局限性**：
- 不同图的特征向量无法直接比较
- 特征向量存在 ±1 模糊性

#### 4.2.2 GraphiT的相对位置编码

**核心创新**：用核函数调制注意力分数

```
标准注意力：
Attention(Q, V) = softmax(Q · Q^T / √d) · V

GraphiT注意力：
PosAttention(Q, V, K_r) = softmax(Q · Q^T / √d ⊙ K_r) · V
                          ↑                            ↑
                        内容相似度                   结构相似度
```

**更新公式**：

```
X = X + D^(-1/2) · PosAttention(Q, V, K_r)
```

### 4.3 子结构编码 (GCKN)

**图卷积核网络 (GCKN)**：

```
核心思想：枚举并编码局部子结构

节点u的特征：
X(u) = Σ ψ(p)    ,  p ∈ P_k(u)
        ─────────    ─────────
        路径嵌入      u的k步内所有路径
```

**路径特征编码**：
```
路径 p = [v, v₁, v₂, ..., vₖ]

特征拼接：
p_features = [node_features(v), node_features(v₁), ...]

通过高斯核嵌入到RKHS
```

### 4.4 完整架构

```
┌─────────────────────────────────────────────────────────────┐
│                     GraphiT 架构                            │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  输入：图 G = (V, E)，节点特征 X                            │
│                                                             │
│  ┌─────────────┐                                           │
│  │ 节点特征增强 │ ← GCKN层输出（可选）                      │
│  └──────┬──────┘                                           │
│         │                                                   │
│  ┌──────▼──────┐                                           │
│  │  Transformer │ ← 核心处理模块                            │
│  │  Layers      │ ← 位置编码 + 自注意力                      │
│  └──────┬──────┘                                           │
│         │                                                   │
│  ┌──────▼──────┐                                           │
│  │ 全局池化     │ ← Mean/Max Pooling                        │
│  └──────┬──────┘                                           │
│         │                                                   │
│  输出：图级别表示 → 分类/回归任务                           │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 5. 实验设计与结果

### 5.1 数据集

| 数据集 | 类型 | 样本数 | 节点特征 | 任务 |
|--------|------|--------|----------|------|
| MUTAG | 分子 | 188 | 7 | 二分类 |
| PROTEINS | 蛋白质 | 1,113 | 3 | 二分类 |
| PTC | 化学 | 344 | 18 | 二分类 |
| NCI1 | 分子 | 4,110 | 37 | 二分类 |
| ZINC | 分子 | 12,000 | 21 | 回归 |

### 5.2 基线方法

| 方法 | 说明 |
|------|------|
| MF | 分子指纹 (Molecular Fingerprint) |
| GCN | 图卷积网络 |
| GAT | 图注意力网络 |
| GIN | 图同构网络 |
| GCKN | 图卷积核网络 |
| Graph Transformer | Dwivedi & Bresson的版本 |

### 5.3 主要结果

#### 分类准确率 / 回归误差

| 方法 | MUTAG | PROTEINS | PTC | NCI1 | ZINC |
|------|-------|----------|-----|------|------|
| MF | 81.5 | 71.9 | 57.3 | 80.6 | 0.387 |
| GCN | 78.9 | 75.8 | 54.0 | 75.9 | 0.367 |
| GAT | 80.3 | 74.8 | 55.0 | 76.8 | 0.384 |
| GIN | 82.6 | 73.1 | 55.0 | 81.7 | 0.387 |
| GCKN-subtree | 87.8 | 72.0 | 62.1 | 79.6 | 0.474 |
| Graph Transformer | 79.3 | 65.8 | 58.4 | 78.9 | 0.359 |
| **GraphiT (GCKN+Adj PE)** | **90.5** | **71.1** | **57.9** | **81.4** | **0.211** |

**关键发现**：
- GraphiT在多个数据集上超越GNNs
- ZINC数据集提升显著（MAE从0.359降至0.211）
- 结合GCKN特征和相对位置编码效果最佳

### 5.4 消融实验

#### 不同位置编码策略对比

| 结构编码 | 无PE | Adj PE | 2-step RW | 3-step RW | Diffusion |
|----------|------|--------|-----------|-----------|-----------|
| 无 | 70.0 | 79.7 | 78.0 | 77.6 | 78.9 |
| LapPE | 74.6 | 78.7 | 78.4 | 78.7 | 77.8 |
| GCKN | 78.1 | **81.4** | 81.0 | 81.0 | 81.0 |

**结论**：
- 相对位置编码优于绝对位置编码（LapPE）
- 结合GCKN特征和相对位置编码效果最佳

## 6. 可视化与可解释性

### 6.1 注意力分数可视化

GraphiT的注意力分数可以揭示重要的分子亚结构。

**案例1：硝基噻酚酰胺-甲基苯**

```
结构：含NO₂基团的芳香化合物

观察：
• 第1-2层：信息从O原子流向N原子
• 第3层：碳骨架的所有原子都"关注"N原子
         ↓
      N是NO₂的关键原子
      NO₂基团已知具有致突变性

预测：正确分类为致突变物
```

### 6.2 关键发现

```
┌─────────────────────────────────────────────────────────────┐
│ 可视化发现：                                                 │
│                                                             │
│ • 注意力分数能够自动识别化学上重要的官能团                    │
│                                                             │
│ • 致突变亚结构（如NO₂、NH₂）被正确识别                       │
│                                                             │
│ • 长程分子内氢键也被模型捕获                                  │
│                                                             │
│ → 为科学应用中的模型可解释性提供新途径                       │
└─────────────────────────────────────────────────────────────┘
```

## 7. 方法优势与局限性

### 7.1 优势

```
✓ 全局信息交互：单层即可实现跨节点通信
✓ 灵活的编码策略：多种核函数可选
✓ 可解释性：注意力分数可揭示重要子结构
✓ 性能优异：在多个基准上超越GNNs
✓ 架构简洁：基于标准Transformer
```

### 7.2 局限性

```
✗ 计算复杂度：自注意力随节点数平方增长
✗ 大图挑战：需要高效Transformer技术
✗ 最优核函数选择：不同任务需要不同核
```

**解决方案**：可结合Longformer、Reformer等高效Transformer

## 8. 总结与展望

### 8.1 主要贡献

1. **将Transformer应用于图数据**
   - 证明了配备适当位置编码的Transformer可以与GNN竞争

2. **基于核函数的相对位置编码**
   - 利用扩散核、随机游走核编码结构相似性
   - 避免绝对位置编码的跨图迁移问题

3. **GCKN子结构特征**
   - 枚举并编码局部路径子结构
   - 显著提升模型性能

4. **可视化机制**
   - 注意力分数可揭示重要的图 motif
   - 为科学应用提供可解释性

### 8.2 未来方向

```
• 大规模预训练：探索类似NLP的预训练-微调范式
• 更高效的注意力：处理大规模图
• 更多子结构类型：循环、graphlets等
• 下游任务扩展：节点分类、链接预测等
```
