GNNs 最大的劣势是节点为中心的局限性，难以捕捉全局信息，尤其是在动态时序图上。传统 GNN（如 MPNN）需要多轮消息传递，动态场景下效率低下，难以实现快速全局理解。

为解决这一问题，需要全局并行感知的中央单元，attention 机制（如 Transformer 的 global token + node/hypergraph tokens）可即时整合全局信息，捕捉时序变化，并将全局反馈高效传递至各个单元。

本研究采用以全局 token 引导节点理解整体变化，central/global 机制加速全局特征获取与动态关系建模，提升预测能力。

我们处理的数据是多个时间切片的动态监控，利用全局监控者从多个时序片段中获取潜在结果。

RouteNet 将网络建模为 link/queue/flow 三类节点的异构超图，效果显著。新模型拟在此基础上重构，理解该三节点超图结构的优势及适用于动态任务的原因。流（flow）体现动态性，queue 和 link 相对静态。RouteNet 的创新是在同一超图中结合动态（流）与结构（q、l）信息，但物理本质仍需关注。

总结两点：1）为何 RouteNet 的三节点异构超图结构有效；2）如何保留其优点，规避其问题。

在形态异构拓扑泛化存在瓶颈，需要探讨瓶颈本质以及如何结合拓扑与物理感知。RouteNet 结构感知有限但擅长物理建模，建议物理相关部分用其复现，结构感知任务引入新模型，并考虑多头结构、分步训练与联合优化。
