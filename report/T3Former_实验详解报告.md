┌─────────────────────────────────────────────────────────────────────────────────────┐
│                                                                                     │
│  为什么交通网络数据集 T3former 提升显著?                                              │
│                                                                                     │
│  ┌─────────────────────────────────────────────────────────────────────────────┐  │
│  │                                                                             │  │
│  │  原因1: 时序拓扑变化模式是关键分类依据                                       │  │
│  │         ─────────────────────────────────────────────────                   │  │
│  │         交通拥堵本质上是道路网络的拓扑结构发生变化:                          │  │
│  │         • 正常: 边权重重，路径选择多样                                       │  │
│  │         • 拥堵: 边权重变轻(限速)，某些路径失效                               │  │
│  │         → T3former 的拓扑描述符(Betti数)能捕获这种变化                       │  │
│  │                                                                             │  │
│  │  原因2: 谱密度(DoS)反映交通流的频率分布                                       │  │
│  │         ─────────────────────────────────────────────────                   │  │
│  │         归一化拉普拉斯特征值分布反映:                                        │  │
│  │         • 连通分量数量(拥塞路段数)                                           │  │
