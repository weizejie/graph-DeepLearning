│   │          │                                                                   │   │
│   │          ▼                                                                   │   │
│   │   ┌──────────────────┐                                                      │   │
│   │   │ self.attn(x, x, x)│  ← 自注意力: Q=K=V=stacked                          │   │
│   │   │ (MultiheadAttn)  │                                                      │   │
│   │   └────────┬─────────┘                                                      │   │
│   │            │                                                                  │   │
│   │            ├──────────────────────────────────────┐                          │   │
│   │            │                                      │                          │   │
│   │            ▼                                      ▼                          │   │
│   │   attn_output (B, 3, D)              attn_weights (B, heads, 3, 3)           │   │
│   │            │                                      │                          │   │
│   │            ▼                                      ▼                          │   │
│   │   reshape → (B, 3*D)              mean(dim=1) → (B, 3, 3)                  │   │
│   │            │                                      │                          │   │
│   │            └──────────────────┬───────────────────┘                          │   │
│   │                               │                                              │   │
│   │                               ▼                                              │   │
│   │                     返回 (fused, weights)                                     │   │
│   │                                                                             │   │
│   └─────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                     │

## 3. 数据流与调用链路

### 3.1 T3Former 完整数据流

```
T3Former.forward() 完整调用链：

forward(x_gsage, edge_index, batch, transformer_input, transformer_input1)
        │
        ├─────────────────────────────────────────────────────────────────────────────┐
        │                              第一阶段：三分支并行计算                        │
        │                                                                             │
        │   ┌─────────────────────────────────────────────────────────────────┐      │
        │   │                                                                  │      │
        │   │   分支1 (Betti特征)           分支2 (DoS特征)         分支3 (图)    │      │
        │   │                                                                  │      │
        │   │   transformer_input           transformer_input1       x_gsage     │      │
        │   │        │                          │                     │          │      │
        │   │        ▼                          ▼                     ▼          │      │
        │   │   ┌──────────────┐          ┌──────────────┐    ┌──────────────┐   │      │
        │   │   │ Transformer  │          │ Transformer  │    │    SAGE      │   │      │
        │   │   │ Classifier   │          │ Classifier   │    │              │   │      │
        │   │   │              │          │              │    │              │   │      │
        │   │   │ ① embedding │          │ ① embedding │    │ ① SAGEConv×N │   │      │
        │   │   │ ② pos_enc   │          │ ② pos_enc   │    │ ② BatchNorm  │   │      │
        │   │   │ ③ Transformer│          │ ③ Transformer│    │ ③ ReLU      │   │      │
        │   │   │   Encoder   │          │   Encoder   │    │ ④ global_   │   │      │
        │   │   │ ④ flatten   │          │ ④ flatten   │    │    pool     │   │      │
        │   │   │ ⑤ fc        │          │ ⑤ fc        │    │              │   │      │
        │   │   └──────┬───────┘          └──────┬───────┘    └──────┬───────┘   │      │
        │   │          │                          │                     │          │      │
        │   │          ▼                          ▼                     ▼          │      │
        │   │        out1 (B, D)            out2 (B, D)            out3 (B, D)    │      │
        │   │        (Betti嵌入)            (DoS嵌入)              (图嵌入)       │      │
        │   │                                                                  │      │
        │   └─────────────────────────────────────────────────────────────────┘      │
        │                                                                             │
        └─────────────────────────────────────────────────────────────────────────────┘
                                         │
                                         ▼
        ┌─────────────────────────────────────────────────────────────────────────────┐
        │                              第二阶段：特征堆叠                              │
        │                                                                             │
        │   stacked = torch.stack([out1, out2, out3], dim=1)                          │
        │                                                                             │
        │   形状变换:                                                                 │
        │   - out1, out2, out3: 各 (B, D)                                            │
        │   - torch.stack 后: (B, 3, D)                                              │
        │                                                                             │
        └─────────────────────────────────────────────────────────────────────────────┘
                                         │
                                         ▼
        ┌─────────────────────────────────────────────────────────────────────────────┐
        │                              第三阶段：注意力融合                            │
        │                                                                             │
        │   attn_out, weights = self.attn_fusion(stacked)                             │
        │                                                                             │
        │   内部流程:                                                                 │
        │   1. attn(stacked, stacked, stacked) → attn_output (B, 3, D)               │
        │   2. attn_output.reshape → (B, 3*D)                                       │
        │   3. attn_weights.mean → (B, 3, 3) 表示分支间相互贡献度                     │
        │                                                                             │
        └─────────────────────────────────────────────────────────────────────────────┘
                                         │
                                         ▼
        ┌─────────────────────────────────────────────────────────────────────────────┐
        │                              第四阶段：分类输出                              │

### 4.2 AttentionFusion 内部交互详解

```
AttentionFusion 注意力权重矩阵的含义：

输入形状: (B, 3, embed_dim)  ← 三分支输出堆叠

┌─────────────────────────────────────────────────────────────────────────────────────┐
│                                                                                     │
│   attn_weights 形状: (B, n_heads, 3, 3)                                            │
│                                                                                     │

│                                                                                     │

│                                                                                     │
│   ┌─────────────────────────────────────────────────────────────────────────────┐   │
│   │                                                                             │   │
│   │              分支j (被关注)                                                  │   │
│   │          Betti    DoS      Graph                                             │   │
│   │       ┌────────┬────────┬────────┐                                          │   │
│   │       │        │        │        │                                          │   │
│   │ Betti │ w_11   │  w_12  │  w_13  │  ← Betti分支给各分支的关注权重             │   │
│   │       │        │        │        │                                          │   │
│   │       ├────────┼────────┼────────┤                                          │   │
│   │  DoS  │ w_21   │  w_22  │  w_23  │  ← DoS分支给各分支的关注权重               │   │
│   │       │        │        │        │                                          │   │
│   │       ├────────┼────────┼────────┤                                          │   │
│   │ Graph │ w_31   │  w_32  │  w_33  │  ← Graph分支给各分支的关注权重               │   │
│   │       │        │        │        │                                          │   │
│   │       └────────┴────────┴────────┘                                          │   │
│   │           │         │         │                                              │   │
│   │       分支i (关注者)                                                          │   │
│   │                                                                             │   │
│   │   对角线 w_ii 表示分支"自关注"程度                                             │   │
│   │   非对角线 w_ij 表示分支i 对 分支j 的关注程度                                  │   │
│   │                                                                             │   │
│   └─────────────────────────────────────────────────────────────────────────────┘   │
│                                                                                     │
